# node-websocket-demo

## 项目搭建

```sh
npm install
```

## 运行项目

由于项目使用 TypeScript 进行编写，需要先构建项目

```sh
npm run build
```

开启一个终端，运行 WebSocket 服务

```sh
npm run server
```

开启另一个终端，运行 WebSocket 客户端

```sh
npm run client
```
